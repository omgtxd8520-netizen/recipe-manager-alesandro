# RecipeManager

Autor: Alesandro David Fajardo Torres

RecipeManager es una aplicación simple para almacenar, buscar y analizar recetas. Incluye:

- API REST mínima (Flask) que usa MongoDB
- Docker Compose con MongoDB y Jupyter Notebook
- Notebooks para análisis y visualización de recetas
- Tests básicos y datos de ejemplo

Características

- Guardar y listar recetas
- Búsqueda por ingredientes y tags
- Notebook Jupyter para explorar la data y probar ML/estadística

Uso rápido

1. Instalar Docker Desktop
2. Ejecutar: `docker compose up --build`
3. Abrir API en http://localhost:5000 y Jupyter en http://localhost:8888
